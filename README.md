# Forras-admin2-react

This project made for second edition of [forras-admin](https://github.com/pzoli/forras-admin)

## Used technologies
- PrimeReact 10.8.4
- idc-client-ts (Keycloak connection) 3.1.0

## install
```bash
npm i
```
## run
```bash
npm run dev
```

 ## Connected projects
 - [keycloak-jwtauth-swagger](https://github.com/pzoli/keycloak-jwtauth-swagger)
